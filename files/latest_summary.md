# 各国政策与监管变化  
### 美国《Bitcoin for America法案》提出  
**总结**：  
1. 美国共和党众议员Warren Davidson提出《Bitcoin for America法案》，核心内容包括允许公民使用比特币缴纳联邦税，并将税收收入注入新设立的“战略比特币储备”。  
2. 法案认为比特币的稀缺性与普及度可提升其价值，作为财富保值工具，同时避免纳税人因使用比特币缴税产生应税收益，储备金由美国财政部长管理。
